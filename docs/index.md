# Nebius Provider

The Nebius AI Cloud provider for Terraform offers a declarative way to manage your Nebius AI Cloud infrastructure as code.

## Getting Started

Use these guides first:

- [Quickstart](guides/quickstart.md)
- [Install the provider](guides/install.md)
- [Authentication](guides/authentication.md)
- [Working with sensitive values](guides/sensitive-values.md)

Service accounts are the recommended authentication method for automation and shared Terraform workflows.

## Example Usage

```terraform
terraform {
  required_providers {
    nebius = {
      source = "nebius/nebius"
    }
  }
}

provider "nebius" {
  service_account = {
    account_id_env       = "NB_SA_ID"
    public_key_id_env    = "NB_AUTHKEY_PUBLIC_ID"
    private_key_file_env = "NB_AUTHKEY_PRIVATE_PATH"
  }
}
```
